## J. 운영 원칙

- **단일 진실 원천(SSoT)**: 같은 내용을 두 곳에 쓰지 않는다. ERD는 D1/E1, 이용약관은 G2만.
- **버전 관리**: 모든 Markdown은 Git 레포 `/docs` 하위 관리. Figma·Notion은 링크만 연결.
- **리뷰 규칙**: SRS·API·보안·법무 문서는 **2인 이상 리뷰** 필수.
- **갱신 주기**: 매 스프린트 종료 시 문서 변경 사항 Changelog에 요약.
- **금지 단어**: 모든 문서·카피·UI에서 **"여행·투어·관광"** 금지 — `G8`에 상세. CI에 `grep` 룰 추가 권장.
