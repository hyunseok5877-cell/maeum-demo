## 0. 디자인 원칙 (Product Principles)

모든 화면·컴포넌트·카피는 아래 6원칙을 통과해야 한다.

1. **여백이 먼저** — 정보밀도 > 여백이면 실패. 럭셔리는 여백이 곧 신뢰.
2. **감정 → 이성** — 사진·영상·스토리로 먼저 감정 후킹, 그 다음 가격·상세.
3. **스크롤은 여정** — 히어로 → 신뢰 → 큐레이션 → 깊이. 한 페이지에서 단계적 몰입.
4. **사람의 흔적** — 큐레이터·후기·수치는 있으나 과시하지 않음. "누가 만들었다"의 온기.
5. **CTA는 대화로** — "예약하기" 같은 거래적 언어 최소화. "문의하기 / 큐레이션 시작 / 담아두기" 같은 관계적 언어.
6. **금지어 철저** — "여행/투어/관광/패키지 투어" 단어 절대 금지. "경험·세션·큐레이션·라운지·프로그램·저니(정신적 의미)"로.

## 2. 홈 페이지 (`/`) — 섹션별 상세

### 구조 원칙
Black Tomato의 **"감정 → 신뢰 → 큐레이션 → 깊이 → 전환"** 흐름을 한국 맥락에 맞게 재구성. 홈은 **10~12개 섹션, 총 스크롤 6~8 뷰포트**.

### 섹션 순서

#### §1. 히어로 (Hero)
- **레이아웃**: 풀스크린 (100vh) — 배경 영상 or 고해상도 사진 슬라이드 (3~5초 크로스페이드)
- **오버레이**: 45% 블랙 그라데이션 (상단→중단)
- **카피 구조**:
  ```
  [작은 라벨]    By Maeum — Curated Experiences
  [메인 헤드]    꿈꾸던 하루를 설계합니다
  [서브]        당신의 마음이 향하는 단 한 번의 경험.
  ```
- **CTA 2개** (나란히):
  - Primary: "경험 둘러보기" → `/experiences`
  - Secondary: "나만의 하루 설계 받기" → `/request-curation`
- **스크롤 힌트**: 하단 중앙 가는 수직선 + 아래 화살표 (애니메이션은 ease-in-out 1.5s 무한루프)

#### §2. 감정 스테이트먼트 (Emotional Statement)
- **레이아웃**: 큰 여백, 한 문장만 중앙.
- **카피 예시**:
  > "모든 경험은 하나의 감정에서 시작됩니다."
- **효과**: 스크롤 진입 시 fade-up (GSAP ScrollTrigger, viewport 진입 시에만)

#### §3. 이 주의 큐레이션 (Featured This Week) — **핵심**
- **레이아웃**: 가로 3~4 컬럼 카드. 큰 이미지 + 간결한 카피.
- **카드 구성**:
  ```
  [대형 사진 (4:5 비율)]
  [지역 태그]  서울
  [타이틀]    한강에서 시작하는 선셋 야경
  [한줄]     요트와 함께 저녁 6시, 단 두 사람을 위한 시간.
  [가격]     from ₩3,200,000   ←  소문자 "from", 달러단위 대신 ₩
  ```
- **주의**: "상품·판매·할인" 단어 금지. "추천"은 OK. 가격 표기 최소화 (숫자 크기는 본문보다 작게).

#### §4. 탐색 방식 선택 (Start Your Journey) — Black Tomato 핵심 패턴
- **레이아웃**: 탭 4개 + 탭별 그리드 8~12개 카드
- **탭 구성**:
  | 탭 | 내용 | 아이템 예시 |
  |---|---|---|
  | **무드별** (By Feeling) | 성향 기반 | 고요 / 짜릿함 / 로맨틱 / 자기보상 |
  | **지역별** (By Region) | 지도 기반 | 서울 / 부산 / 제주 (+ Coming Soon 그림자 카드 강원·경기) |
  | **월별** (By Month) | 계절·시즌 | 4월 벚꽃 · 6월 요트 · 10월 단풍 |
  | **특별 기획** (In the Spotlight) | 편집 컬렉션 | 프로포즈 레디 / 주말 이스케이프 / 1인 리트릿 |
- **상호작용**:
  - 탭 전환 시 그리드 crossfade (150ms)
  - 카드 hover: 이미지 scale 1.02, 오버레이 가볍게

#### §5. 성향 테스트 인바이트 (Quiz Invitation)
- **레이아웃**: 좌측 카피 + 우측 일러스트(또는 컬러 블록) 2컬럼
- **카피**:
  ```
  [타이틀]    당신이 원하는 건 어떤 하루인가요?
  [본문]     6개의 질문으로 당신만의 경험 유형을 찾고,
             큐레이터의 맞춤 제안을 받아보세요.
  [CTA]      3분 테스트 시작 →
  ```
- **디자인**: 다른 섹션과 대비되는 오프화이트 → 옅은 세피아 배경으로 구분

#### §6. 큐레이터 소개 (The Curators)
- **레이아웃**: 작은 포트레이트 3~5개 + 이름 + 전문 분야
- **의도**: "사람이 만든다"는 신뢰감 (Black Tomato의 설립자 비디오 역할)
- **예시**:
  ```
  [사진] 김OO — 슈퍼카 드라이빙 전문
  [사진] 박OO — 요트 & 해상 큐레이션
  [사진] 이OO — 외승 & 프라이빗 리트릿
  ```
- Phase 1엔 오너 1인 + 가상의 큐레이터 1~2명(실제 협업 큐레이터) 소개

#### §7. 후기 (Whispers)
- **레이아웃**: 가로 슬라이더, 카드 1~3개 동시 노출
- **카드**:
  ```
  "어느 곳에서도 받을 수 없던 시간이었다."
  — J, 서울, 2026.03
  [경험 태그]  요트 선셋
  ```
- 익명 표기 + 이니셜 + 지역/월 조합 (럭셔리 특유의 절제된 후기 톤)
- Phase 1은 시드 후기 3~5건으로 시작 (과장 없이 사실 기반)

#### §8. 지역 탐색 (Explore by Region)
- **레이아웃**: 구글맵 임베드 + 마커 인터랙션
- **기능**: 마커 클릭 시 사이드 카드에 해당 지역 경험 3~5개 노출
- **모바일**: 지도 대신 수직 지역 리스트 + 대표 이미지

#### §9. 철학·브랜드 (Our Approach)
- **레이아웃**: 좌측 에세이 3단락 + 우측 이미지
- **카피 방향**: "왜 우리가 만드는가 / 어떻게 다른가 / 무엇을 약속하는가"

#### §10. 자주 묻는 질문 (간소)
- 3~5개만. 더보기는 `/faq`로 이동
- "여행" 단어 포함 시 자동 경고 (CI grep)

#### §11. 하단 CTA (Final Invitation)
- **레이아웃**: 풀폭 세로 60vh, 큰 카피 + CTA 1개
- **카피**:
  ```
  당신의 마음이 향하는 단 하나의 하루,
  함께 설계하겠습니다.

  [큐레이션 시작하기 →]
  ```

#### §12. 푸터
- 사업자 정보 (전자상거래법 표기 의무) — 법인명·대표·주소·사업자번호·통신판매업 신고번호·이메일
- **중개자 고지**: "당사는 통신판매 중개자입니다. 계약 당사자는 제휴 공급사입니다."
- SNS 링크 (인스타·유튜브)
- 뉴스레터 구독 (이메일 1개 필드)

## 4. 카탈로그 페이지 (`/experiences` 및 하위)

### 상단
- 카테고리 헤더 (대형 타이틀 + 서브카피 + 히어로 이미지 1장, 40vh)
- 예: `/experiences/supercar` →
  ```
  Supercar
  도심의 아드레날린, 단 몇 시간의 지배자.
  ```

### 필터 바
- **좌측 필터**: 지역 / 무드 / 인원 / 예산 / 시간대
- **우측 정렬**: 추천 / 최신 / 가격 낮은순 / 평점 높은순
- 필터 변경 시 URL 쿼리 동기화 (`?region=seoul&mood=thrill`)

### 카드 그리드
- 3열 (데스크톱) / 2열 (태블릿) / 1열 (모바일)
- 카드 가로세로비 4:5, 가운데 대형 사진
- 하단에 지역·카테고리·제목·가격 from `₩X,XXX,XXX`

### 페이지 하단
- Pagination 대신 "더 보기" 버튼 (한 번에 12개씩 추가 로드)

## 5. 경험 상세 페이지 (`/experiences/[slug]`)

### §1. 히어로 갤러리
- 좌측 대형 이미지 (9:5) + 우측 스택된 3장 (모바일은 스와이프)
- 영상 있으면 썸네일 → 클릭 시 모달 재생

### §2. 제목·요약·CTA 바 (sticky)
- 제목 / 한 줄 요약 / 지역 · 인원 · 소요시간 · 가격 from
- 우측: "담아두기(하트)" + "문의하기(primary)"
- 스크롤 내려도 상단 80px sticky bar 유지

### §3. 이야기 (Story)
- 에디토리얼 톤 본문 (3~5단락, 큰 폰트, 긴 행간)
- 큐레이터 한마디 (인용 블록)

### §4. 포함/불포함 (Included / Excluded)
- 2컬럼 체크리스트
- 아이콘은 그로테스크 1px stroke

### §5. 시간표·일정 샘플
- 타임라인 컴포넌트 (14:00 도착 → 14:30 출발 → 17:00 귀환 등)

### §6. 조건·주의 (Considerations)
- 최소·최대 인원 / 날씨 취소 정책 / 복장 / 차량 조건 / 반려동물 여부

### §7. 옵션 (Add-ons)
- 옵션 목록 체크박스 + 추가 가격
- 선택하면 상단 CTA 바 가격 자동 갱신

### §8. 유사 경험 (You May Also Like)
- 카드 3~4개

### §9. 후기
- 3~5개 표시 + 더보기

### §10. 최종 CTA
- 대형 "문의하기 / 예약 시작" 버튼

## 6. 성향 테스트 (`/quiz`)

### 구조
- 6문항 × 각 4지선다
- 한 화면에 한 문항 (풀스크린), 하단 progress bar
- 응답 즉시 다음 문항 자동 전환 (150ms)
- 뒤로가기 허용
- 비회원 가능 (세션 토큰으로 결과 저장)

### 결과 페이지
- 유형명 (영문 + 한글) + 유형 대표 이미지
- 유형 설명 3~4줄
- "이 유형에게 맞는 경험" 3개 카드
- CTA: "맞춤 큐레이션 받기" → `/request-curation` (답변 자동 프리필)
- 공유 버튼 (카카오·링크복사) — **개인정보 포함 X**

## 7. 역제안 문의 (`/request-curation`)

### 폼 단계 (3스텝)
1. **무엇을 원하시나요?**
   - 날짜 범위, 지역, 인원, 예산 범위, 무드 체크박스
2. **누구와 함께하시나요?**
   - 연인 / 가족 / 친구 / 비즈니스 / 혼자
3. **자유롭게 들려주세요**
   - 자유 서술 (1500자)
   - 파일 첨부 (영감이 된 레퍼런스 이미지 등) — 선택

### 제출 후
- 확인 페이지: "3일 내 큐레이터가 3안 제안서를 보내드립니다"
- 이메일 자동 발송 (접수 확인)
- 비회원도 가능 (이메일·이름·전화 필수)

## 8. 챗봇 (RAG 컨시어지)

### 진입
- 우하단 원형 FAB (부유 버튼) — 단색, 아이콘 심플
- "안녕하세요. 어떤 하루를 꿈꾸고 계세요?" 인사 말풍선

### UI
- 하단에서 올라오는 슬라이드 패널 (데스크톱은 우측 420px, 모바일은 풀스크린)
- 메시지 버블: 사용자(우 다크) vs AI(좌 오프화이트)
- AI 답변 내 경험 제안은 **카드 형태**로 인라인 삽입 (이미지·제목·가격·"상세" 버튼)
- 타이핑 인디케이터 (3개 점 fade loop)

### 기능
- 빠른답변 버튼 (초기 화면 3개): "서울에서 주말", "제주 프로포즈", "슈퍼카 드라이브"
- "큐레이터에게 넘기기" 버튼 — 항상 노출. 클릭 시 `/request-curation`으로 답변 프리필 이동
- 음성 입력 (모바일) — Phase 2

### 금기
- 할인·환불·가격 변경 **임의 약속 금지** (시스템 프롬프트 하드룰)
- "여행"·"투어" 단어 생성 금지 (정규식 후처리 필터)
- 답변 내 외부 링크 생성 제한 (자사 URL만)

## 9. 모션·인터랙션 가이드

| 상황 | 규칙 |
|---|---|
| 섹션 진입 애니메이션 | GSAP ScrollTrigger (뷰포트 진입 시 실행, 페이지 로드 시 X) |
| 이미지 호버 | scale 1.02, 200ms ease-out |
| 버튼 호버 | 배경 0.08 어두워짐, 150ms |
| 페이지 전환 | Next.js default + 상단 진행바 (2px, 300ms) |
| 스크롤 | lenis smooth-scroll (데스크톱만, 모바일은 native) |
| 무빙 헤로 | video autoplay muted loop, Poster image, `preload=metadata` |
| 로딩 스피너 | 얇은 단색 원형 회전, 400ms 이상 지연 시에만 노출 |
| 픽셀 단위 스냅 | 섹션 사이 스냅 스크롤 X (가이드라인 위반 — 자연 스크롤) |

## 10. 접근성 (A11y) 체크리스트

- 모든 이미지 `alt` 필수
- 대비비 4.5:1 이상 (WCAG AA)
- 포커스링 `outline` 명시 (브라우저 기본 제거하지 않기)
- 키보드 Tab 순서 논리적 (헤더 → 본문 → 푸터)
- `aria-label` 아이콘 버튼에 모두 부여
- 영상 자동재생 시 `muted` 필수, 자막 옵션 제공
- 모션 민감 사용자: `prefers-reduced-motion` 존중 → 애니메이션 정적 치환

## 11. 카피 스타일 가이드 (Copy Tone)

### ✅ 권장
- 경험·세션·큐레이션·라운지·프로그램·저니(정신적)·순간·하루
- 문장은 짧게. 접속사 최소.
- 숫자·가격은 본문보다 작게.
- 2인칭 "당신" 사용 OK, 반말 X.
- 감각동사: 머물다·닿다·새기다·설계하다·담아두다

### ❌ 금지
- 여행·투어·관광·패키지투어·상품·할인·특가·이벤트·대박·꿀팁
- 이모지 남용
- 과장된 수식어 ("최고의", "유일한", "극강의")
- "예약하기"/"구매하기" → "문의하기"/"시작하기"로 대체

### 샘플 문구 교체
| 나쁨 | 좋음 |
|---|---|
| 여행 상품 예약 | 경험 문의 시작 |
| 인기 투어 TOP 5 | 이 주의 큐레이션 |
| 할인 이벤트 | 한정 초대 |
| 가이드 동행 | 큐레이터 동반 |
| 여행 일정표 | 세션 타임라인 |
