## 3. 수익 모델 검증 (리서치 기반)

| 항목 | 수치 | 근거 |
|---|---|---|
| 플랫폼 커미션 (업계 표준) | 20-30% | DMCQuote·Fora Travel |
| 럭셔리 카테고리 커미션 | 25-35% | 동 |
| 객단가 300만원 × 25% | **75만원/건 수수료** | 계산 |
| 월 1건만 → **연 900만원**(최저선, 1인 월세 커버 불가) | 현실 | 계산 |
| **손익분기 역산** (추정) | 회원 300명 × 연 4건 × 300만원 × 25% = **연 9억원** | 04.G-10 |

**→ "월 1건만 돼도 좋다"는 유저 포지셔닝은 브랜드 철학(exclusivity)으론 맞지만, 실제 손익은 회원 300명급 스케일이 필요.** 이 간극을 멤버십 연회비(50~500만원)로 메우는 2트랙이 현실적.

### 제안 수익 모델 v1
```
(1) 경험 중개수수료:     객단가 × 25%   = 75만원/건
(2) 연간 멤버십 (Gold):  연 100만원
(3) 연간 멤버십 (Black): 연 500만원 (매니저 전담·시크릿 경험·우선예약)
(4) 풀커스텀 설계비:     건당 50~200만원 (역제안 큐레이션)
```

## 5. 🚨 즉시 대응 필요한 법적 리스크 (최우선)

### 5.1 "여행" 단어 오염 상태
**현재 상태**: 유저가 직전 대화에서 **"어떤 여행을 좋아하세요?"** 설문 문구를 사용. 이대로 UI에 박히면 **여행업 무등록 3년 이하 징역/3천만원 벌금 리스크** (관광진흥법 §82).

**대응**:
- 모든 설문·카피·챗봇 프롬프트에서 "여행→경험" 일괄 치환
- CI grep 룰 추가 (금지어 커밋 차단)
- `03.G8` 금지어 가이드를 **킥오프 문서 첫 장에 박제**

### 5.2 통신판매중개업 구조 확정
**핵심 4개 동시 충족 (공정위 재분류 방지)**:
1. 결제 화면·이용약관에 **"당사는 통신판매 중개자, 계약 당사자는 공급사"** 명시
2. PG 결제 시 **실제 판매자(공급사) 표시**
3. 매출은 **중개수수료만 인식** (GMV ≠ 매출)
4. **숙박·항공 결합 상품 금지** (즉시 여행업 영역으로 재분류)

### 5.3 소개팅·슈퍼카 고위험 공급 정책
- 소개팅: **실명·신원인증 KYC** 의무, 성범죄경력 조회 고지, NDA 표준
- 슈퍼카: 정식 렌트업자만, **자차·대물·대인 한도 명시 보험 확약서** 필수 (불법 렌트 역사 있음, 04.C-3)

## 10. 최종 권고

**Go / No-Go 판단: GO (조건부)**

✅ **Go 근거**
- 시장 CAGR 22% (한국 experiential luxury)
- 빈 시장 검증 완료 (독립형 럭셔리 경험 큐레이션 부재)
- 유저 가설이 Knight Frank/McKinsey/Bain 리포트로 뒷받침됨
- 해외 모델 (Indagare/Quintessentially/Black Tomato) 수익성 입증
- 우리 차별화(RAG+MBTI+블랙골드 브랜드) 해외에도 완성형 없음

⚠️ **Go의 조건**
1. 첫 2주 내 법무 자문 + 통신판매중개업 신고
2. "여행" 단어 전면 제거
3. 공급 파트너 3곳 LOI 확보 후 프로덕트 개발 시작 (supply-first)
4. 초기 객단가·운영강도 유지할 **회원 Cap 설정** (베타 100명 한정)
5. 소개팅 카테고리는 Phase 2로 연기

**이 5개 조건을 지키면 12-18개월 내 연매출 10억원급은 보수적으로 가능.**
