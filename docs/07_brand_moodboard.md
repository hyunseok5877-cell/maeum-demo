## 0. 한 줄 포지션

> **"쉬운 럭셔리가 아니라, 고요한 럭셔리."**
> 샤넬 블랙골드의 과시를 걷어내고, Aesop·Jil Sander·Rimowa의 "기능과 여백으로 말하는 럭셔리"를 한국적 정서의 '마음'이라는 단어로 풀어낸다.

## 1. 브랜드 무드 5원칙

| # | 원칙 | 의미 |
|---|---|---|
| 1 | **여백이 장식이다** | 빽빽한 디자인은 곧 싸구려. 여백은 비용. 비용이 곧 럭셔리. |

## 4. 타이포그래피

### 4.1 폰트 선택

| 역할 | 1순위 (유료) | 2순위 (무료 대체) | 비고 |
|---|---|---|---|
| Display (대형 헤드) | **Canela** (Commercial Type) | **EB Garamond** (Google) | 세리프. 모던 버전 |
| Display 대체 | **Reckless Neue** | **Libre Caslon Display** | 더 도도한 톤 원할 때 |
| Body / UI | **Inter** (무료) + **Pretendard** (한글, 무료) | — | 산세리프. 가독성 |
| Numeric | **Inter Tabular** | — | 가격·예약번호 tabular-nums |

**권고**: Phase 1은 무료 조합으로 시작. `EB Garamond + Pretendard + Inter`.

### 4.2 타입 스케일 (Tailwind 커스텀)

| 토큰 | 폰트 | 크기 | 행간 | letter-spacing | 용도 |
|---|---|---|---|---|---|
| `display-xl` | Serif | 72px | 1.05 | -0.02em | 히어로 헤드라인 |
| `display-lg` | Serif | 56px | 1.08 | -0.02em | 섹션 헤드 |
| `display-md` | Serif | 40px | 1.12 | -0.01em | 서브섹션 |
| `title` | Sans | 24px | 1.3 | -0.005em | 카드 제목 |
| `body-lg` | Sans | 18px | 1.6 | 0 | 본문 롱폼 |
| `body` | Sans | 16px | 1.6 | 0 | 본문 기본 |
| `body-sm` | Sans | 14px | 1.5 | 0 | 메타데이터 |
| `caption` | Sans | 12px | 1.4 | 0.04em (UPPERCASE) | 라벨·배지 |

### 4.3 타이포 규칙

- 한 페이지에 **3단계 이상의 위계 금지** (Display / Title / Body)
- 헤드라인 줄바꿈은 의미 단위로 수동 제어 (`<br/>` OK)
- 대문자 라벨은 `caption` 사이즈만, letter-spacing 0.04em
- 숫자(가격·인원·시간)는 `font-variant-numeric: tabular-nums`
- 한글은 Pretendard 필수. Noto Sans KR 사용 시 letter-spacing -0.01em 보정

## 5. 이미지·영상 톤

### 5.1 이미지 원칙

| 체크 | 기준 |
|---|---|
| 색온도 | **웜 쪽으로 +200K** (쌀쌀한 푸른빛 금지) |
| 노출 | 약간 overexposed OK, underexposed 금지 |
| 포커스 | 피사체 주변 bokeh 허용. 경험(사물) 주인공 |
| 보정 | 톤 매핑 약하게. 강한 HDR·tilt-shift 금지 |
| 종횡비 | 카드 4:5, 히어로 16:9, 상세 갤러리 혼합 |
| 해상도 | 최소 1920px 긴쪽, 히어로는 2560px+ |
| 사람 등장 | **얼굴 클로즈업 자제**. 손·뒷모습·실루엣 중심 (개인정보·모델권) |

### 5.2 영상 원칙

| 체크 | 기준 |
|---|---|
| 길이 | 히어로 루프 8~15초, 상세는 최대 60초 |
| 음성 | 기본 muted, 사용자가 활성화 선택 |
| 코덱 | H.264 MP4 + WebM 동시 제공 |
| 모션 | 카메라 천천히 글라이드, 급한 줌 금지 |
| 색보정 | LUT 통일 (Phase 1 말에 전용 LUT 제작) |

### 5.3 아이콘·일러스트

- 아이콘: **1px stroke 그로테스크** (Phosphor Regular · Iconoir · Heroicons Outline)
- 일러스트: **선화 최소주의**. 캐릭터·의인화 금지
- 3D 금지. 애니메이션 일러스트 금지.

## 6. 그리드·여백·레이아웃

### 6.1 그리드

- 데스크톱: **12 컬럼 · gutter 24px · max-width 1440px**
- 태블릿: 8 컬럼 · gutter 20px
- 모바일: 4 컬럼 · gutter 16px · 사이드 margin 20px

### 6.2 수직 리듬

섹션 간 `padding-y` 기본 **120px (데스크톱)** / 80px (태블릿) / 64px (모바일).
**절대 64px 미만으로 줄이지 않기** (여백 원칙 위반).

### 6.3 정렬

- **좌정렬이 기본**. 중앙정렬은 히어로·대형 CTA·인용문에만
- 우정렬은 숫자·가격·날짜 정도에만

## 7. 컴포넌트 스타일 가이드

### 7.1 버튼

| 타입 | 배경 | 텍스트 | 높이 | radius | 호버 |
|---|---|---|---|---|---|
| Primary | `#1A1A1A` | `#FAFAF8` | 52px | 0 (직각) | 배경 #000 |
| Secondary | 투명 | `#1A1A1A` | 52px | 0 | 배경 `#F2F0EA` |
| Ghost (링크형) | 투명 | `#1A1A1A` | auto | 0 | 밑줄 fade-in |
| Pill (태그) | `#F2F0EA` | `#1A1A1A` | 32px | 999 | 배경 `#E6E4DE` |

모든 버튼 transition `180ms ease-out`.

### 7.2 카드

- 배경 `#FFFFFF`, 테두리 `1px solid #E6E4DE` 또는 **테두리 없음 + 하단만 1px 라인**
- 그림자 **기본 없음**. 필요시 `0 1px 2px rgba(0,0,0,0.04)`
- 내부 padding `24px`
- 이미지 위·텍스트 아래 레이아웃. 이미지 상단이 카드 상단 flush

### 7.3 폼

- Input: 하단 1px 밑줄만 (border-bottom), 좌우·상 테두리 없음
- Focus: 밑줄 2px → 검정
- Label: 캡션 사이즈 + 대문자 + letter-spacing 0.04em, 입력 위에 배치
- Placeholder: Mist Grey

### 7.4 네비게이션

- 고정 헤더 높이 72px. 스크롤 감지 시 56px로 축소 (200ms)
- 배경: 투명 → 스크롤 > 40px 시 `rgba(250,250,248,0.92) + backdrop-blur(12px)`
- 메뉴 항목: caption 사이즈 + UPPERCASE + letter-spacing 0.08em

## 8. 로고 방향 (v0.1 제안)

### 옵션 A — 한글 로고타입만
```
  마    음
 MAEUM · CURATED EXPERIENCES
```
- 한글 "마음"을 EB Garamond 느낌의 한글 세리프(Noto Serif KR) 대형
- 아래에 소문자 "MAEUM" + 중간 dot + 설명 서브타이틀

### 옵션 B — 한글·영문 병기
```
마음 / MAEUM
```
- 기울기 없는 슬래시로 병기, 같은 무게의 세리프

### 옵션 C — 워드마크 + 심볼
```
⟁  MAEUM
```
- 작은 추상 심볼(삼각형·원 변형) + 워드마크
- 심볼은 앱 아이콘·파비콘 겸용

**v0.1 권장**: **A안** (한글 정서 강조 + 해외 확장 시 B로 확장 용이).

**로고 색상**: 잉크 블랙 단색만. 금색·장식 금지.

## 9. 사운드·보이스 (선택)

- UI 사운드는 Phase 1 미사용 (럭셔리 무드 훼손 위험)
- 브랜드 영상에 나레이션 추가 시 — 저음 여성 혹은 침착한 남성. 한국어 또렷하게. 외국 모델 기용 지양
- BGM은 로파이·앰비언트 피아노 쪽 (Nils Frahm · Ryuichi Sakamoto 계열). 비트 강한 음악 금지

## 10. 사례 — 홈 히어로 비주얼 시뮬레이션

```
배경: 선셋 한강 요트 위에서 촬영한 영상 (8초 루프)
    - 카메라 슬로우 팬, 좌→우
    - 색감 웜 +200K
    - 풀 블랙 오버레이 40% top-down 그라데이션

텍스트 (좌정렬, 하단 30% 지점):

  BY MAEUM — CURATED EXPERIENCES        ← caption / 흰색 / tracking 0.08em
                                         ← (여백 24px)
  꿈꾸던 하루를                            ← Display XL / Serif / 흰색
  설계합니다.                              ← (줄바꿈 의미 단위)
                                         ← (여백 32px)
  당신의 마음이 향하는 단 한 번의 경험.    ← body-lg / Sans / 흰색 85% 투명
                                         ← (여백 48px)
  [경험 둘러보기] [나만의 하루 설계 받기]  ← Primary + Secondary 버튼

스크롤 힌트: 우측 하단 얇은 수직선 40px + 아래 화살표
```

## 11. 금지 리스트 (Don'ts)

- ❌ 금색 그라데이션
- ❌ 이모지 (UI·카피 전부)
- ❌ 볼드 + 이탤릭 동시 사용
- ❌ 중앙정렬 폼
- ❌ 풀블랙 배경 (장례식 느낌) → Obsidian으로 부드럽게
- ❌ 형광 액센트 컬러
- ❌ 3D 그래픽·뉴모피즘·글래스모피즘
- ❌ 3개 이상 컬러 동시 사용
- ❌ 경쟁적 카피 ("No.1", "최고", "1위")
- ❌ 이벤트·할인 배너 (럭셔리 무드 파괴)

## 12. 체크리스트 (컴포넌트·페이지 승인 기준)

배포 전 본 체크리스트 통과 필수.

- [ ] 색상은 팔레트 내에서만 사용했는가
- [ ] 폰트는 세리프(Display) + 산세리프(Body) 2종만인가
- [ ] 섹션 간 수직 간격 최소 64px 확보했는가
- [ ] 이미지 포맷 WebP/AVIF, 해상도 충분한가
- [ ] 영상 muted·loop·poster 설정했는가
- [ ] 버튼 호버·포커스 상태 있는가
- [ ] 접근성 대비비 4.5:1 이상인가
- [ ] 모바일 375px에서 레이아웃 깨짐 없는가
- [ ] "여행·투어·관광" 단어 포함 0건인가
- [ ] 이모지·할인·이벤트 단어 0건인가
